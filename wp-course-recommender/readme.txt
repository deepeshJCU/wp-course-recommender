A simple rule-based course recommendation plugin for students based on their interests.

== Description ==
This plugin helps students find suitable courses based on their interests. Administrators can define mappings between student interests and courses using a simple JSON-based input system in the WordPress admin panel.

== Features ==
Rule-Based Course Recommendation: Suggests courses based on predefined interest mappings.
Admin Panel for Configuration: Easily manage interest-to-course mappings.
Secure Input Validation: Ensures valid and secure user inputs.
Shortcode Support: Embed the recommendation form anywhere using `[course_recommender]`.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Course Recommender' in the admin panel to set up mappings.
4. Add the shortcode `[course_recommender]` to any page or post.

== Frequently Asked Questions ==
= How do I configure interest-to-course mappings? =
Go to the plugin settings under the WordPress admin panel and enter mappings in JSON format.

= Can I customize the form layout? =
Yes, you can override the form styles using custom CSS in your theme.

= Does this plugin store student data? =
No, this plugin does not store any user data permanently.
